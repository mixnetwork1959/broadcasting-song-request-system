# broadcasting-song-request-system
A modern multi-station song request system for RadioBOSS, RadioDJ and AzuraCast.


# Broadcasting Song Request System

## Features

- Multi-station support
- Song search
- Song request queue
- Request cooldown
- Admin dashboard
- Responsive design
- Connector-based architecture
- RadioBOSS support
- RadioDJ support
- AzuraCast support
